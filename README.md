如何製作Line半垢/半人機 :
------
- `pkg update`
- `pkg install git`
- `pkg install python2`
- `pip2 install rsa`
- `pip2 install thrift==0.9.3`
- `pip2 install requests`
- `pip2 install bs4`
- `pip2 install gtts`
- `pip2 install beautifulsoup`
- `pip2 install googletrans`
- `git clone https://github.com/ningxue520/ningxue`
- `cd ningxue`
- `python2 ningxue.py`

如何再次啟動半垢 (主機重啟後) :
------
- `cd ningxue`
- `python2 ningxue.py`
