# 半垢製作

-  製作Line半垢:
- `pkg update`
- `pkg install git`
- `pkg install python2`
- `pip2 install rsa`
- `pip2 install thrift==0.9.3`
- `pip2 install requests`
- `pip2 install bs4`
- `pip2 install gtts`
- `pip2 install beautifulsoup`
- `pip2 install googletrans`
- `git clone https://github.com/austin0916/austinline`
- `cd austinline`
- `python2 austin.py`
-  再次啟動半垢 (主機重啟後) :
- `cd austinline`
- `python2 austin.py`

-  取得Line代碼 (TOKEN) :
-  https://github.com/austin0916/linetoken`
